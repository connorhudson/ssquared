S^2 v1.0

A block module and local server for Snap! to interface with the Parallax S2.

Requirements

  • A Scribbler robot
  • Python 2.6+
  • The myro module for Python.

Getting Started

  • Load Snap! in your browser window.
  • Import the S^2.xml file.
  • In a command line, navigate to the directory where you are storing these files.
  • Type python S^2.py.
  • Follow the instructions onscreen.
  • The program should say something like serving on port 3610. Go ahead and launch Snap! in your browser window.
  • Enjoy!
